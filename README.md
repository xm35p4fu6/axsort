# AXSORT

> install boost
``` sudo apt-get install libboost-all-dev ```

> install google test (some command need sudo)
```
apt install libgtest-dev
cd /usr/src/gtest
cmake CMakeLists.txt
make
cp *.a /usr/lib
```

> run sample test
```
$ mkdir build
$ cd build
$ cp ../Makefile .
$ make sample
$ ./sample
```
